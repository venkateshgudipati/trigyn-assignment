Hi,

This assignment having two folders 
1.backend
2.frontend

to run above application we need  to install following items on your macine
1. Node Js
2. MongoDB
3. Angular CLI

----------------------------- BACK END ----------------------------------------------------------

1. open command prompt 
2. cd backend
3. npm install
4. node index.js

----------------------------- BACK END ----------------------------------------------------------

----------------------------- Front END ----------------------------------------------------------

1. open command prompt
2. cd frontend
3. npm install
4. ng serve --port 4300
5. open browser http://locahost:4300
